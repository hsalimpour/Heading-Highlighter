# هایلایت تگ‌های H1 تا H6 برای بررسی ساختار هدینگ و سئو صفحات وب (Heading Highlighter)

![Manifest](https://img.shields.io/badge/Manifest-V3-orange)
![Feature](https://img.shields.io/badge/Feature-H1--H6%20Heading%20Highlight-success)
![SEO](https://img.shields.io/badge/SEO-Helper-brightgreen)
![License](https://img.shields.io/badge/License-MIT-green)

**Heading Highlighter** یک افزونه سبک و کاربردی برای Chrome است که تگ‌های هدینگ (`H1` تا `H6`) را در هر صفحه وب به‌صورت **رنگی و برچسب‌دار** نمایش می‌دهد تا بتوانید ساختار هدینگ‌ها را سریع بررسی کنید و مشکلات رایج سئو (مثل چند H1 یا ترتیب نادرست هدینگ‌ها) را در یک نگاه تشخیص دهید.

---

## چرا Heading Highlighter؟
اگر سئوکار، تولیدکننده محتوا، طراح سایت یا برنامه‌نویس هستید، این افزونه کمک می‌کند:
- تعداد و محل هدینگ‌های **H1 تا H6** را سریع ببینید
- ترتیب هدینگ‌ها را بررسی کنید (H1 → H2 → H3 …)
- ایرادهای ساختاری محتوا را بدون Inspect یا View Source پیدا کنید
- ساختار صفحه را برای بهینه‌سازی سئو و تجربه کاربری بهتر تحلیل کنید

---

## امکانات افزونه
- پشتیبانی کامل از **H1 تا H6**
- هایلایت رنگی هر تگ با رنگ اختصاصی
- نمایش لیبل شناور روی هر هدینگ (H1/H2/…)
- اجرای سریع فقط روی تب فعال
- حذف کامل هایلایت‌ها و بازگشت به حالت عادی با یک کلیک
- بدون جمع‌آوری داده، بدون ردیابی، بدون ارسال اطلاعات

---

## رنگ‌بندی تگ‌های هدینگ (H1 تا H6)
این افزونه برای هر تگ هدینگ یک رنگ مشخص در نظر گرفته است:

| تگ | رنگ | کد رنگ |
|---|---|---|
| H1 | قرمز تیره | `#ae0000` |
| H2 | آبی | `#0051ffff` |
| H3 | زرد | `#f2af2f` |
| H4 | سبز | `#01a511ff` |
| H5 | بنفش | `#a910c4ff` |
| H6 | فیروزه‌ای | `#00bfdcff` |

> این رنگ‌ها مطابق تنظیمات فعلی کد افزونه هستند.

---

## نحوه استفاده
1. وارد هر صفحه وب شوید
2. روی آیکن افزونه کلیک کنید
3. دکمه **Highlight Headings** را بزنید تا هدینگ‌ها رنگی و برچسب‌دار شوند
4. برای بازگرداندن صفحه به حالت عادی، روی **Clear Highlights** کلیک کنید

---

## نصب به‌صورت دستی (Developer Mode)
1. ریپازیتوری را دانلود یا Clone کنید
2. در Chrome آدرس زیر را باز کنید:
   - `chrome://extensions`
3. گزینه **Developer mode** را فعال کنید
4. روی **Load unpacked** کلیک کنید
5. پوشه پروژه را انتخاب کنید

---

## مجوزها (Permissions)
این افزونه از مجوزهای زیر استفاده می‌کند:
- `activeTab` و `scripting`  
برای اجرای هایلایت فقط روی **تب فعال** و تنها زمانی که شما دکمه‌ها را می‌زنید.

---

## حریم خصوصی (Privacy)
این افزونه:
- هیچ اطلاعاتی ذخیره نمی‌کند
- هیچ داده‌ای ارسال نمی‌کند
- هیچ ردیابی یا آنالیتیک ندارد
- فقط روی همان صفحه‌ای که باز کرده‌اید و با کلیک شما اجرا می‌شود
